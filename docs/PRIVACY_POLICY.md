| Right | Description | How to Exercise |
|-------|-------------|-----------------|
| **Access** | Get a copy of your data | Settings > Privacy > Export Data |
| **Rectification** | Correct inaccurate data | Settings > Edit Profile |
| **Erasure** | Delete your data | Settings > Delete Account |
| **Portability** | Receive data in machine-readable format | Settings > Privacy > Export Data |
| **Restriction** | Limit processing | Contact privacy@bodaconnect.ao |
| **Objection** | Object to processing | Contact privacy@bodaconnect.ao |
